- Like photos

# Installation
Follow the steps bellow to install:
```git
1. Clone the repository
2. Navigate to .\fronted\
3. Run 'npm i'
4. Navigate to .\backend\
5. Run 'npm i'
6. Create a .env file
7. define the .env
```
<strong>.env example:</strong>
```env
PORT=APP_PORT
DB_USER=DATABASE_USERNAME
DB_PASS=YOUR_PASSWORD
JWT_SECRET=YOUR_SECRET
```
